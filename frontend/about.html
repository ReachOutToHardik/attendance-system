About This Project

A complete teacher-side attendance management system

Project Overview

Attendance System is a web-based application for teachers to efficiently mark, manage, and review student attendance in any school, college, or organization. It features a clean, minimal UI and integrates with Google Sheets and Google Cloud for secure data storage, reporting, and backup. The system is designed for scalability and can be used for any class, section, or subject.

Features

  • Mark Attendance: Teachers can select class, section, subject, and date, then mark students present or absent with a simple interface.
  • Attendance Reports: View and download detailed attendance records by class, section, subject, and month. Data is synced with Google Sheets and Google Cloud for backup and sharing.
  • Student Data Import: Easily manage student lists using PDF or text files.
  • Modern UI/UX: Minimal, responsive design for fast, distraction-free workflow.
  • Google Sheets & Google Cloud Integration: Attendance is automatically updated in a connected Google Sheet and stored securely in Google Cloud for transparency, backup, and easy access.
  • Scalable & Customizable: Works for any college, department, or subject. Easy to extend for more features.

How It Works

  1. Mark Attendance: Go to the Mark Attendance page, select class/section/subject/date, and mark students present. Submit to save and sync data.
  2. View Report: Go to the Report page, select class/section/subject/month, and view/download attendance records.
  3. Data Sync: All attendance is stored locally and pushed to Google Sheets and Google Cloud for backup and reporting.

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js (Express)
  • Database: JSON files (local), Google Sheets & Google Cloud (cloud)

Folder Structure

  • frontend/ - UI pages and assets
  • backend/ - Node.js server and data logic
  • studentslist/ - Student data import files

How to Use

  1. Clone or download the repository.
  2. Install backend dependencies with npm install in the backend folder.
  3. Start the backend server with node server.js.
  4. Run python -m http.server 5500 in the frontend folder to start a local web server.
  5. Open http://localhost:5500 in your browser to use the Attendance System frontend.
  6. Why? Modern browsers block some features (like AJAX/fetch requests) when opening HTML files directly. Running a local server ensures all features work correctly and allows the frontend to communicate with the backend.
  7. Configure Google Sheets API credentials in backend/credentials.json.
